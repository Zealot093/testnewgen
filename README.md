## Description

Алгоритм филтрации курсов по цене работает следующим образом:
При получении данных на вход запускается цикл, в каждой итерации которого проверяется, входят ли верхняя и нижняя границы цены в заданный пользователем диапазон, и в случае выполнения условия курс добавляется в результирующий массив.
